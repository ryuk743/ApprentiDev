Les Cookies en PHP

1. Qu'est-ce qu'un cookie ?

Un cookie est un petit fichier stocké sur l'ordinateur de l'utilisateur par le navigateur web. Il permet de sauvegarder des informations entre les visites d'un site (identifiant, préférences, etc.).

2. Créer un cookie en PHP

Utilisez la fonction setcookie() pour créer un cookie : 

<?php
setcookie("nom", "Jean", time() + 3600); // expire dans 1 heure
?>

Le cookie sera disponible lors du prochain chargement de page.

3. Lire un cookie

Les cookies sont accessibles via le tableau $_COOKIE : 

<?php
echo $_COOKIE["nom"];
?>

4. Supprimer un cookie

Pour supprimer un cookie, il suffit de le réécrire avec une date d'expiration passée : 

<?php
setcookie("nom", "", time() - 3600);
?>

5. Bonnes pratiques

  • Définir les cookies avant tout affichage HTML (avant <html>).
  • Utiliser les options de sécurité (secure, httponly, samesite).
  • Ne stockez jamais d'informations sensibles dans un cookie.
