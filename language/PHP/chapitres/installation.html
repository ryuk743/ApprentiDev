Guide d'installation de PHP

Un tutoriel complet pour configurer PHP sur votre système Windows

Installation de PHP sur Windows

1. Télécharger PHP

Rendez-vous sur le site officiel de PHP : php.net.
Cliquez sur "Downloads" puis sur "Windows downloads".
Choisissez la version correspondant à votre système (64 bits ou 32 bits).
Prenez de préférence la version Non Thread Safe si vous débutez. Cliquez pour télécharger le fichier .zip.

2. Extraire les fichiers

Une fois le fichier téléchargé, extrayez-le dans un dossier sur votre disque (par exemple : C:\php).

3. Ajouter PHP aux variables d'environnement

  1. Ouvrez le menu Démarrer, tapez "variables d'environnement" et ouvrez l'option correspondante.
  2. Dans la section "Variables système", trouvez la variable Path et cliquez sur "Modifier".
  3. Ajoutez le chemin du dossier où se trouve PHP (ex. : C:\php).
  4. Cliquez sur "OK" pour enregistrer.

4. Vérifier l'installation

Ouvrez l'invite de commande (CMD) et tapez la commande suivante : 

php -v

Si PHP est installé, la version s'affichera. Sinon, un message d'erreur concernant une DLL manquante peut apparaître.

5. Installer Visual C++ Redistributable

Si une erreur du type "MSVCR140.dll is missing" s'affiche, cela signifie qu'il faut installer le Visual C++ Redistributable.
Téléchargez-le depuis le site de Microsoft : Télécharger Visual C++ Redistributable
Installez la version correspondant à votre système (x64 ou x86), puis redémarrez le terminal.

6. Utiliser PHP

Une fois PHP installé, vous pouvez commencer à créer et exécuter des fichiers .php depuis votre terminal.

Astuce : Pour tester rapidement PHP, créez un fichier test.php avec le code suivant : 

<?php
phpinfo();
?>

Exécutez-le avec la commande php test.php pour voir toutes les informations sur votre installation PHP.
