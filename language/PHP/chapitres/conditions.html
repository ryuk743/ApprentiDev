Les Conditions en PHP

Prendre des décisions dans votre code

Introduction aux conditions

Les conditions permettent d'exécuter du code seulement si certaines conditions sont remplies. Elles sont essentielles pour créer une logique dans vos programmes.

// Exemple basique
$note = 12;
if ($note >= 10) {
    echo "Bravo, vous avez la moyenne !";
} else {
    echo "Dommage, vous n'avez pas la moyenne.";
}

Structure de base

1. La condition simple (if/else)

if (condition) {
    // Code exécuté si la condition est vraie
} else {
    // Code exécuté si la condition est fausse
}

2. Conditions multiples (elseif)

if ($note > 10) {
    echo "Au-dessus de la moyenne";
} elseif ($note == 10) {
    echo "Juste la moyenne";
} else {
    echo "En dessous de la moyenne";
}

Opérateurs de comparaison

Opérateur Description Exemple
== Égal à (valeur) $a == $b
=== Identique (valeur et type) $a === $b
!= Différent (valeur) $a != $b
!== Non identique (valeur ou type) $a !== $b
< Inférieur à $a < $b
> Supérieur à $a > $b
<= Inférieur ou égal $a <= $b
>= Supérieur ou égal $a >= $b

Important : Préférez === et !== pour éviter les conversions implicites.

Opérateurs logiques

ET logique (&& ou and)

// Le magasin est ouvert entre 9h-12h ET 14h-17h
if (($heure >= 9 && $heure <= 12) || ($heure >= 14 && $heure <= 17)) {
    echo "Le magasin est ouvert";
}

OU logique (|| ou or)

// Action utilisateur : 1 (attaque), 2 (défense) ou 3 (passe)
if ($action == 1 || $action == 2 || $action == 3) {
    // Action valide
}

NON logique (!)

if (!isset($variable)) {
    // La variable n'existe pas
}

Structure switch

Alternative aux multiples elseif quand on teste une même variable contre plusieurs valeurs.

switch ($action) {
    case 1:
        echo "J'attaque";
        break;
    case 2:
        echo "Je défends";
        break;
    case 3:
        echo "Je passe mon tour";
        break;
    default:
        echo "Commande inconnue";
}

Exercices pratiques

Exercice 1 : Vérification d'âge

$age = (int)readline("Quel est votre âge ? ");
if ($age >= 18) {
    echo "Vous êtes majeur";
} elseif ($age > 0) {
    echo "Vous êtes mineur";
} else {
    echo "Âge invalide";
}

Exercice 2 : Horaires d'ouverture

$heure = (int)readline("Quelle heure est-il ? ");
if (($heure >= 9 && $heure < 12) || ($heure >= 14 && $heure < 17)) {
    echo "Le magasin est ouvert";
} else {
    echo "Le magasin est fermé";
}

Bonnes pratiques

  • Utilisez === plutôt que == sauf besoin spécifique
  • Convertissez les entrées utilisateur avec (int), (float) ou (string)
  • Privilégiez la lisibilité avec une indentation claire
  • Utilisez des parenthèses pour clarifier les opérations complexes
  • Préférez switch pour tester une variable contre plusieurs valeurs constantes
  • Commentez les conditions complexes
