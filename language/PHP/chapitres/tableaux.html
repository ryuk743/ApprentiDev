Les Tableaux en PHP

Organiser et manipuler des collections de données

Introduction aux tableaux

Les tableaux (arrays) permettent de stocker plusieurs valeurs dans une seule variable. Ils sont essentiels lorsqu'on doit gérer des collections de données comme des listes de notes, des informations d'utilisateurs, etc.

// Problème avec plusieurs variables
$note1 = 10;
$note2 = 15;
$note3 = 12;
// ... devient ingérable avec beaucoup de notes

// Solution : utiliser un tableau
$notes = [10, 15, 12];

Création de tableaux

Il existe deux syntaxes pour créer des tableaux :

1. Tableaux indexés (liste simple)

// Syntaxe moderne (recommandée)
$notes = [10, 15, 12];

// Ancienne syntaxe (toujours valable)
$notes = array(10, 15, 12);

2. Tableaux associatifs

$eleve = [
    'nom' => 'Dupont',
    'prenom' => 'Marc',
    'notes' => [10, 15, 12]
];

Accès aux éléments

Dans un tableau indexé

$notes = [10, 15, 12];
echo $notes[1]; // Affiche 15 (index commence à 0)

Dans un tableau associatif

$eleve = ['nom' => 'Dupont', 'prenom' => 'Marc'];
echo $eleve['prenom']; // Affiche "Marc"

Modification des tableaux

Modifier une valeur existante

$eleve['prenom'] = 'Jean'; // Remplace 'Marc' par 'Jean'

Ajouter un élément

// Dans un tableau indexé
$notes[] = 16; // Ajoute en fin de tableau

// Dans un tableau associatif
$eleve['classe'] = 'CM2'; // Ajoute une nouvelle clé

Tableaux multidimensionnels

Un tableau peut contenir d'autres tableaux :

$classe = [
    [
        'nom' => 'Dupont',
        'prenom' => 'Jean',
        'notes' => [16, 12, 15]
    ],
    [
        'nom' => 'Martin',
        'prenom' => 'Sophie',
        'notes' => [14, 18, 13]
    ]
];

// Accès à la 2ème note du 1er élève
echo $classe[0]['notes'][1]; // Affiche 12

Fonctions utiles pour les tableaux

  • count($tableau) - Nombre d'éléments
  • array_push($tableau, $valeur) - Ajoute en fin
  • array_pop($tableau) - Retire le dernier élément
  • array_merge($tab1, $tab2) - Fusionne des tableaux
  • sort($tableau) - Tri ascendant

Affichage des tableaux

// echo ne fonctionne pas bien pour les tableaux
$notes = [10, 15, 12];
echo $notes; // Affiche "Array"

// Utiliser plutôt print_r() ou var_dump()
echo '<pre>';
print_r($notes);
echo '</pre>';

// Affiche :
// Array
// (
//     [0] => 10
//     [1] => 15
//     [2] => 12
// )

Exercices pratiques

Exercice 1 : Calcul de moyenne

$notes = [10, 15, 12];
$moyenne = array_sum($notes) / count($notes);
echo "Moyenne : " . round($moyenne, 2);

Exercice 2 : Gestion d'élèves

$eleves = [
    ['nom' => 'Dupont', 'notes' => [10, 12]],
    ['nom' => 'Martin', 'notes' => [15, 18]]
];

foreach ($eleves as $eleve) {
    $moyenne = array_sum($eleve['notes']) / count($eleve['notes']);
    echo $eleve['nom'] . " : " . round($moyenne, 1) . "<br>";
}

Bonnes pratiques

  • Utilisez la syntaxe courte [] plutôt que array()
  • Nommez clairement vos clés associatives
  • Indentez correctement les tableaux multidimensionnels
  • Commentez les structures complexes
  • Vérifiez l'existence d'une clé avec isset() avant d'y accéder
