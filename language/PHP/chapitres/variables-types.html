Débuter avec PHP

Création d'un fichier PHP

Pour travailler avec PHP, créez un fichier avec l'extension .php (ex. : demo.php).

Ce fichier peut être ouvert dans un éditeur de texte comme Visual Studio Code, Sublime Text ou PHPStorm.

Bon à savoir : Un fichier PHP peut contenir du HTML, du CSS et du JavaScript en plus du code PHP.

Structure de base d'un fichier PHP

Le code PHP doit être écrit entre les balises :

<?php
// Code PHP ici
?>

Le texte hors des balises est directement affiché tel quel.

Variantes : PHP supporte aussi les balises courtes <? ?> mais elles ne sont pas recommandées pour la portabilité.

Utiliser le terminal

Depuis le terminal intégré dans Visual Studio Code, exécutez un fichier PHP avec la commande :

php nom_du_fichier.php

Options utiles :

  • -f pour spécifier un fichier : php -f script.php
  • -r pour exécuter du code directement : php -r 'echo "Hello";'
  • -a pour le mode interactif

Les variables en PHP

Une variable commence toujours par un $, suivi du nom de la variable.

$nom = "Marc";        // Chaîne de caractères
$age = 25;            // Entier
$prix = 19.99;        // Nombre flottant
$estActif = true;     // Booléen

Conventions de nommage :

  • Doit commencer par une lettre ou underscore
  • Ne peut pas commencer par un chiffre
  • N'autorise que caractères alphanumériques et underscores
  • Est sensible à la casse ($Nom et $nom sont différents)

Afficher une variable

On utilise echo ou print pour afficher une variable :

echo $nom;          // Affiche la valeur
print $nom;         // Alternative à echo
var_dump($nom);     // Affiche le type et la valeur (débogage)

Différence : echo peut prendre plusieurs paramètres, print retourne toujours 1.

Concaténation de chaînes

Pour coller deux chaînes, on utilise le symbole . :

echo $prenom . " " . $nom;  // Marc Dupont

Depuis PHP 8.0, on peut aussi utiliser l'interpolation avec des accolades :

echo "{$prenom} {$nom}";    // Marc Dupont

Différence entre guillemets simples et doubles

  • Guillemets doubles :
    • Interprètent les variables ("Bonjour $nom")
    • Interprètent les caractères spéciaux (\n, \t)
    • Plus lents à traiter que les simples
  • Guillemets simples :
    • N'interprètent pas les variables ('Bonjour $nom' affichera littéralement "$nom")
    • N'interprètent que \' et \\
    • Plus rapides à traiter
$nom = "Jean";
echo "Bonjour $nom";    // Bonjour Jean
echo 'Bonjour $nom';    // Bonjour $nom

Types de base en PHP

Type Exemple Description
Entiers $x = 10; Nombres entiers, positifs ou négatifs
Flottants $y = 12.5; Nombres à virgule flottante
Chaînes $nom = "Jean"; Séquences de caractères
Booléens $valide = true; Vrai ou faux
Null $inconnu = null; Variable sans valeur
Tableaux $tab = [1, 2, 3]; Collection ordonnée de valeurs

Vérification de type : Utilisez is_int(), is_string(), etc. pour vérifier un type.

Exercices pratiques

Exercice 1 : Calcul de moyenne

Afficher la phrase suivante : "Bonjour Marc Dos, vous avez eu 15 de moyenne."

$prenom = "Marc";
$nom = "Dos";
$note1 = 10;
$note2 = 20;
$moyenne = ($note1 + $note2) / 2;

echo "Bonjour $prenom $nom, vous avez eu $moyenne de moyenne.";

Version avec concaténation :

echo "Bonjour " . $prenom . " " . $nom . ", vous avez eu " . $moyenne . " de moyenne.";

Exercice 2 : Conversion de température

Convertir 100°F en Celsius en utilisant la formule : C = (F - 32) * 5/9

$fahrenheit = 100;
$celsius = ($fahrenheit - 32) * 5/9;
echo "$fahrenheit°F équivaut à " . round($celsius, 2) . "°C";

Bonnes pratiques

  • Toujours initialiser les variables avant de les utiliser
  • Utiliser des noms descriptifs ($nombre_articles plutôt que $na)
  • Éviter les variables globales quand c'est possible
  • Utiliser le camelCase ou les underscores pour les noms composés
  • Commenter votre code quand la logique est complexe
